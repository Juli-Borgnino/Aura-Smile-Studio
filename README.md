## 📋 Descripción

SmileCare es un sitio web para una clínica odontológica familiar ficticia. La estética fue diseñada para transmitir calidez y confianza, con una paleta verde esmeralda y tipografía amigable. Incluye secciones completas de servicios, equipo profesional y sistema de turnos.

## 📁 Páginas incluidas

- **Inicio** — Hero con ilustración SVG, servicios, equipo y testimonios
- **Servicios** — Odontología general, infantil, ortodoncia y limpieza
- **Nosotros** — Equipo y filosofía del consultorio
- **Contacto** — Formulario de turnos con validación, horarios y ubicación

## ⚙️ Tecnologías

- HTML5 semántico
- CSS3 — sistema de diseño con variables, mobile-first
- JavaScript puro — menú mobile, scroll animations, validación de formularios

## ✨ Características

- Ilustraciones SVG inline (sin imágenes externas)
- Paleta de colores verde esmeralda + celeste + amarillo
- Tipografía Fraunces + Nunito (Google Fonts)
- Totalmente responsive
- Botón de contacto por WhatsApp
- Animaciones al hacer scroll con IntersectionObserver
- Accesibilidad y SEO básico
